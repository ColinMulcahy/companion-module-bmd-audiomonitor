# Blackmagic Audio Monitor

This module will connect to Blackmagic Design Audio Monitor.

## Configuration

- Type in the device IP address.
- The device will communicate over port 9996.

## Actions

- Audio Meter
- Audio Input
- Speaker Volume
- Headphone Volume
- Mute
- Solo

## Feedbacks

- Audio Meter
- Audio Input
- Speaker Volume
- Headphone Volume
- Mute
- Solo
